Privacy Policy

Last updated: January 9, 2024  ·  View source on GitHub

Introduction

Base64 Image Converter is an open-source tool hosted at mrlipx.github.io/base64-image-converter. This policy explains exactly how the tool handles your data — in short: it doesn't. Everything stays in your browser.

Data Processing

Local processing only

  • All image conversions are processed entirely in your browser using standard Web APIs
  • Images are decoded to Blob objects and previewed via URL.createObjectURL() — no data is transmitted
  • Blob URLs are revoked via URL.revokeObjectURL() when no longer needed
  • You can verify this by opening the browser Network tab — zero requests during conversion

No storage

  • We do not store any images, Base64 data, or conversion history anywhere
  • There is no server-side processing — no server is involved at all
  • All data exists temporarily in your browser's memory and is discarded when you navigate away

No personal data

  • We do not collect any personal information
  • No account, login, or registration is required or available
  • No email addresses, names, IP addresses, or identifiers are collected by this tool

Open Source

Every line of code running in your browser is publicly visible. Privacy isn't a promise here — it's verifiable code.
  • MIT Licensed — free to use, modify, and redistribute
  • GitHub: github.com/MrLiPx/base64-image-converter
  • You can fork and self-host your own instance at any time

Analytics & Cookies

  • We use only GitHub Pages built-in traffic stats — aggregated, anonymous page-view counts
  • No third-party analytics (no Google Analytics, Mixpanel, etc.)
  • We set no cookies of our own; GitHub Pages may set essential infrastructure cookies
  • No tracking or advertising cookies

Security

  • No server-side processing means no server-side risk for your image data
  • Site served over HTTPS via GitHub Pages with industry-standard TLS
  • Open-source code means security issues can be identified and fixed publicly

Compliance

Because we collect no personal data, the compliance footprint is minimal by design. This policy is intended to be compatible with GDPR, CCPA, PIPEDA, and other applicable regulations.

Changes

Changes are committed to the public GitHub repository with full history. The "last updated" date above reflects the most recent revision.
